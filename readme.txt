Running this project for beginners:
You have to run it in a local server, Xampp or Wamp could do the work.
You have to create a database by the name of (jobportal) you can change the database settings in .env file.
Than you have to run the migration command on cli or terminal window => php artisan migrate.
You have to run the command => php artisan create:admin , will create a super admin for you with this credentials=>
User Name: admin@jobportal.com
Password: Job@12345.


There You go. You are good to go.
